Comp Transforma Informação Computação Dados

Na computação real, um dado isolado (como o número 40) não significa nada. O papel do computador é processar dades dentro de um contento para gerar informação atil (como "a temperatura atual é 40°C"). Ense processo é o que chamamos de Ciclo de Dados.

OS tres Pilares do Processamento

Para que um sistema computacional funcione de forma coerente, ele depende de três camadas fundamentals operando em conjunte:

  • Hardware: A infraestrutura fisica (processadores, chips, cabos),
  • Software: As instruções lógicas e códigos que dizem ao hardware o que fazer.
  • Userware: O componente humano que opera o sistema e define o objetivos.
